# Base de Conhecimento de Jogos 

Este é um projeto de front-end que funciona como uma base de conhecimento interativa para jogos. A aplicação exibe uma lista de jogos a partir de um arquivo de dados local e permite que o usuário pesquise e filtre os resultados em tempo real.

##  Funcionalidades
-   **Listagem Dinâmica:** Carrega e exibe uma lista de jogos a partir de um arquivo `data.json`.
-   **Busca em Tempo Real:** Filtra os jogos instantaneamente conforme o usuário digita no campo de busca.
-   **Critérios de Busca Abrangentes:** A busca funciona por nome do jogo, gênero ou desenvolvedor.
-   **Links Externos:** Cada card de jogo contém um link "Saiba mais" que direciona para o site oficial.
-   **Design Responsivo:** A interface se adapta a diferentes tamanhos de tela, de desktops a dispositivos móveis.

##  Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias:

-   **HTML5:** Para a estrutura semântica da página.
-   **CSS3:** Para a estilização, utilizando Flexbox, variáveis CSS e Media Queries para responsividade.
-   **JavaScript (ES6+):** Para a interatividade, manipulação do DOM e a lógica de busca e renderização.
-   **Fetch API:** Utilizada para carregar os dados dos jogos de forma assíncrona.
